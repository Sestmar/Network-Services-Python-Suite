# Network Services Suite

Herramientas de administración y prueba de red desarrolladas en Python con interfaz gráfica (Tkinter).

## Funcionalidades
1. **Cliente SMTP:** Envío de correos HTML autenticados mediante servidor SMTP (probado con Mailtrap).
2. **NetworkLab:**
   - **DNS Resolver:** Resolución directa e inversa de nombres de dominio a IP.
   - **NTP Client:** Sincronización y consulta de hora con relojes atómicos.

## Tecnologías
- Python 3.x
- Librerías: `smtplib`, `socket`, `ntplib`, `tkinter`.
